# DisplayBalance

## 项目简介
这是一个用于显示余额的Python项目。

## 安装
1. 克隆项目
```bash
git clone https://github.com/yourusername/DisplayBalance.git
cd DisplayBalance
```

2. 创建虚拟环境
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
.\venv\Scripts\activate  # Windows
```

3. 安装依赖
```bash
pip install -r requirements.txt
```
